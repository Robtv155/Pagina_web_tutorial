    El elemento header representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación. Un elemento header generalmente contiene: uno o más elementos de encabezado (<h1> - <h6>) logotipo o ícono información sobre la autoría. Nota: Puedes tener varios elementos header en un documento HTML. Sin embargo, un header no puede colocarse dentro de un footer, address u otro elemento header.

  • Encabezados
